🛠 Before You Begin

Prerequisites: The Toolkit

Lock and load your dev environment before we summon a new project.

💻

Modern OS

macOS, Windows (WSL2 recommended), or Linux.

🐘

PHP 8.2+

Statamic loves the latest PHP features. Make sure you're running 8.2 or newer.

🎼

Composer

The heartbeat of your PHP dependency management. Must be globally installed.

The Process

Three Steps to Running

1
🚀 Step 1

Arm Your Terminal

Install the Statamic CLI — your magic wand for generating new sites in seconds. Run this in your terminal:

bash
$ composer global require statamic/cli
Pro Tip

Once it's done, type statamic list to see all the superpowers now at your disposal.

2
🏗 Step 2

Create Your Masterpiece

Navigate to your projects folder and pull a brand-new Statamic instance out of thin air:

macOS/Linux: cd ~/sites Windows: cd C:\dev
bash
$ statamic new my-awesome-site
🎨 Starter Kit

Get a head start with pre-built templates, layouts, and components.
⚡ Blank Canvas

Total creative control from the ground up. Zero assumptions made.

3
🌐 Step 3

Bring It to Life

Statamic is built on Laravel — a built-in server is already waiting. Move into your new directory and fire it up:

bash
$ cd my-awesome-site
$ php artisan serve
http://127.0.0.1:8000
Your Statamic site is live. Open this in your browser.
